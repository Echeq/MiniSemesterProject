# AGENTS.md - TaskFlow AI Instructions

## Quick Start for Workers

When you type `/init` in OpenCode, this file is loaded automatically.

## Available Skill

**Skill:** `ai-log-generate`
**Location:** `ai-skills/skills/ai-log-generate.md`

## How to Use

After using AI to write code, type:

@ai-log


## What Happens

1. The AI asks you for:
   - What you asked the AI
   - What the AI responded
   - What code changed

2. The AI creates a log file in `docs/log/`

3. The log file name looks like: `AI-LOG-2025-01-15-143022-LAPTOP-JOHN.txt`

## Setup (One Time)

Run these commands in your terminal:

```bash
mkdir -p docs/log
mkdir -p ai-skills/skills
mkdir -p ai-skills/scripts
