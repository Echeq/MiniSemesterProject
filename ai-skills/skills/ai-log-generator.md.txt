# AI Log Generator Skill

## Description
This skill generates log files for every AI interaction. Each log is saved in `/docs/log/` with format: `AI-DOC-YYYYMMDD-HHMMSS-PCNAME.txt`

## How to Use This Skill

### Step 1: Create the scripts

Save these files in your project:

**Linux/Mac:** `scripts/new-ai-log.sh`
**Windows:** `scripts/new-ai-log.ps1`

### Step 2: Make script executable (Linux/Mac only)

```bash
chmod +x scripts/new-ai-log.sh
