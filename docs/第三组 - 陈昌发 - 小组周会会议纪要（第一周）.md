## 五、存在问题与风险

### 🔴 关键问题

| 问题 | 影响 | 建议方案 |
|:----|:-----|:---------|
| 缺少 `project_members` 关联表 | 无法按项目精细控制成员权限 | 新建迁移，创建 junction table + RLS |
| 优先级系统未实现（P0-P3） | 无法区分任务紧急程度 | 新增 `priority` 列 + 前端选择器 |
| 标签系统未实现 | 无法分类和过滤任务 | 新建 `labels` + `task_labels` 表 |
| 任务依赖系统未实现 | 无法表达任务间阻塞关系 | 新建 `task_dependencies` 表 + 校验逻辑 |

### 🟡 次要问题

| 问题 | 影响 | 建议方案 |
|:----|:-----|:---------|
| RLS 未按 `project_id` 过滤 | 成员可以看到所有项目任务 | 修改 RLS 策略增加 `project_id` 检查 |
| 无项目设置页面 | 无法编辑项目名称/描述 | 新建 ProjectSettings 组件 |
| 无项目颜色/图标 | 项目列表无视觉区分度 | 增加 `color`/`icon` 列 + 选择器 UI |
| 无提醒/通知系统 | 用户不会收到截止日期提醒 | 后续迭代补充 |

- `xlsx` 已移除（从未使用）
