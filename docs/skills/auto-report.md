# auto-report

Interactive report generator. Collects user input, inserts content into a Markdown template, checks export dependencies, and outputs to `/docs/auto-report/`.

> **Trigger:** `@auto-report`

## Quick Start
1. Run `@auto-report` to start interactive report generation
2. Follow prompts for title, content, and template
3. Report is saved to `docs/auto-report/`

**Example:** `@auto-report` → interactive session

## Description

Walks the user through report creation with a template-based system. Supports multiple template formats and checks for export dependencies.

## Usage

| Command | Action |
|---|---|
| `@auto-report` | Interactive report generation |
| `@auto-report --templates` | List available templates |
| `@auto-report --history` | Show past reports |
| `@auto-report --config` | Configure report settings |
