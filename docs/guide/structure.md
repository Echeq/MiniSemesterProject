## Key directories

| Path | Role |
|---|---|
| `frontend/src/api/` | Supabase client initialization |
| `frontend/src/components/` | React components (Board, Column, TaskCard, Sidebar, Topbar, etc.) |
| `frontend/src/hooks/` | Custom hooks (useAuth, useBoard, useProfile, useProjects, etc.) |
| `frontend/src/locales/` | i18n translation files (en, es, zh, id, ar) |
| `frontend/tests/` | Vitest tests (components/, hooks/, integration) |
| `supabase/migrations/` | Database migration files (source of truth for schema) |
| `docs/api.md` | Supabase API reference |
| `docs/architecture.md` | Architecture & data flow |
| `docs/database.md` | Schema, RLS, migrations |
| `docs/setup.md` | Setup guide |
| `docs/deploy.md` | Deploy guide |
| `docs/guide/` | User guides (ai.md, structure.md, user-manual.md) |
| `docs/meta/` | Project tracking (CHANGES, task, daily-tracking, requirements, todo) |
| `docs/skills/` | AI skill documentation (generated) |
| `docs/reference/` | Technical reference (performance, API collection) |
| `docs/log/` | AI interaction logs |
| `docs/` | All project documentation (see [INDEX.md](/docs/INDEX.md)) |
