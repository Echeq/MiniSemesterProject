## Objective
Stop members from being able to drag cards on desktop and access the mobile move menu.

## Files
- `frontend/src/components/TaskCard.jsx` — disable useSortable for non-admin
- `frontend/src/components/Column.jsx` — thread canDrag prop
- `frontend/src/components/Board.jsx` — pass canDrag + gate mobile menu

## Steps
1. Add `canDrag` prop to TaskCard, disable `useSortable` when false
2. Thread `canDrag` through Column
3. Pass `canDrag={isAdmin}` from Board to both Column renderings
4. Gate mobile menu status-move buttons by `isAdmin`
